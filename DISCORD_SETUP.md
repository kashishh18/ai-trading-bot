# 🤖 Discord Bot Setup Guide\n\nGet cool trading alerts sent directly to your Discord server! Takes 2 minutes to setup.\n\n## Step 1: Create a Discord Server (if you don't have one)\n1. Open Discord\n2. Click the \"+\" button on the left sidebar\n3. Choose \"Create My Own\" → \"For me and my friends\"\n4. Name it \"AI Trading Bot\" or whatever you like\n\n## Step 2: Create a Webhook\n1. In your Discord server, click the gear icon ⚙️ next to your server name\n2. Go to \"Integrations\" in the left menu\n3. Click \"Webhooks\" → \"New Webhook\"\n4. Name it \"AI Trading Bot\" 🤖\n5. Choose which channel to send alerts to (create a #trading-alerts channel)\n6. Click \"Copy Webhook URL\" - this is what you need!\n\n## Step 3: Add Webhook to Your Bot\n1. In your `backend` folder, copy `.env.example` to `.env`:\n   ```bash\n   cp .env.example .env\n   ```\n\n2. Edit the `.env` file and paste your webhook URL:\n   ```bash\n   DISCORD_WEBHOOK_URL=https://discord.com/api/webhooks/1234567890/your-webhook-token-here\n   ```\n\n3. Install the required package:\n   ```bash\n   pip install aiohttp\n   ```\n\n## Step 4: Test It!\n1. Run your trading bot:\n   ```bash\n   cd backend\n   python main.py\n   ```\n\n2. You should see a startup message in your Discord channel! 🚀\n\n## What You'll Get\n\n### 📈 Trade Alerts\n```\n🤖 AI Trading Bot\n📈 Trade Executed!\nBOUGHT AAPL\n💰 Price: $150.25\n📊 Shares: 50  \n💵 Total Amount: $7,512.50\n🧠 AI Confidence: 78.0%\n⏰ Time: 14:32:15\n```\n\n### 🎯 Opportunity Alerts\n```\n🤖 AI Trading Bot  \n🎯 New Trading Opportunities Found!\nAI discovered 3 potential trades\n\n📈 #1: TSLA\nBUY • 85.0% confidence\nExpected: +5.40%\n\n📈 #2: MSFT  \nBUY • 72.0% confidence\nExpected: +3.20%\n```\n\n### 💰 Position Closed Alerts\n```\n🤖 AI Trading Bot\n💰 Position Closed: AAPL\nPROFIT: +$245.75\n📊 Entry Price: $150.25\n📊 Exit Price: $155.17\n📈 Shares: 50\n📅 Days Held: 3 days\n📋 Reason: Take profit triggered\n```\n\n### 📊 Daily Summary\n```\n🤖 AI Trading Bot\n📈 Daily Portfolio Summary  \nPortfolio is UP today\n💼 Total Value: $105,780.50\n📊 Daily Change: +$2,340.75 (+2.25%)\n🎯 Active Positions: 8 stocks\n⚡ Trades Today: 3 executed\n```\n\n## Troubleshooting\n\n**❌ \"Discord notifications disabled\"**\n- Make sure your webhook URL is in the `.env` file\n- Check that the URL starts with `https://discord.com/api/webhooks/`\n\n**❌ \"Discord notification failed: 404\"** \n- Your webhook URL might be wrong\n- Make sure you copied the full URL from Discord\n\n**❌ \"No module named 'aiohttp'\"**\n- Install it: `pip install aiohttp`\n\n## Advanced: Multiple Channels\n\nYou can create different webhooks for different types of alerts:\n\n```bash\n# In your .env file\nDISCORD_TRADES_WEBHOOK=https://discord.com/api/webhooks/.../...     # For trade alerts\nDISCORD_OPPORTUNITIES_WEBHOOK=https://discord.com/api/webhooks/.../... # For opportunities  \nDISCORD_ERRORS_WEBHOOK=https://discord.com/api/webhooks/.../...     # For errors\n```\n\nThen modify the `discord_bot.py` to use different webhooks for different alert types.\n\n## Security Notes\n\n⚠️ **Keep your webhook URL secret!** Anyone with this URL can send messages to your Discord server.\n\n✅ **Best practices:**\n- Don't commit your `.env` file to GitHub\n- Only share webhook URLs with people you trust\n- You can regenerate webhook URLs anytime in Discord settings\n\n---\n\n🎉 **That's it!** Your trading bot will now send awesome alerts to Discord. Enjoy watching your money grow in real-time! 💰🚀
